OpenSea Developer Platform - The first and largest NFT marketplace

As the leading marketplace for Non-Fungible TokensNon-Fungible Tokens - A token that is unique among peers. While Fungible tokens are equivalent and interchangeable (like Ether) Non-Fungible Tokens are unique and distinct, like deeds of ownership, or collectibles. and Semi-Fungible Tokens, OpenSea provides a first-in-class developer platform consisting of an API, SDK, and developer tutorials. Feel free to browse around and get acclimated with developing smart contracts and interacting with NFT data.

Fetch assets with our API

OpenSea provides a REST API for fetching non-fungible token assets based on a set of query parameters. Wallets and websites can now display:

Assets

Get assets filtered by collection, type, or belonging to a particular wallet address

Activity

Get events happening on all assets (not just on our marketplace), including sales, listings, bids, transfers, and more

Accounts

Get usernames and profile details for account addresses
